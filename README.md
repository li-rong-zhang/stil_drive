# stil_drive

基于 Qt 的超精密光学表面轮廓仪驱动与采集软件，集成 STIL CCS Prima 色散共焦传感器与 Delta Tau Power PMAC 多轴运动控制器，实现硬件同步"飞拍"扫描与表面面形分析。

## 项目简介

`stil_drive` 是一款运行于 Windows 平台的桌面应用程序，用于驱动 **STIL CCS Prima 色散共焦传感器** 对光学/机械精密表面（如透镜、反射镜）进行非接触式高精度轮廓测量。

系统核心亮点是**硬件位置同步采集**（"飞拍"技术）：
- **Delta Tau Power PMAC** 运动控制器在预设位置间隔处通过 Gate3 EQU 寄存器输出硬件触发脉冲
- **STIL 传感器** 每收到一个触发脉冲精确采集一个高度测量值
- 软件通过脉冲计数还原每个数据点的绝对空间坐标

这种方式消除了软件定时误差，确保数据点的空间位置精度与运动控制器编码器分辨率一致。

## 功能特性

| 功能 | 描述 |
|------|------|
| **传感器连接** | 通过 USB 自动扫描并连接 STIL CCS Prima 传感器 |
| **多种扫描模式** | 支持 X 轴线性扫描、C+X 轴螺旋扫描、C 轴纬度圆弧扫描 |
| **硬件同步触发** | 通过 TCP/Telnet 向 PMAC 发送 Gate3 EQU 触发脚本，实现硬件级位置同步 |

| **CSV 数据记录** | 可选自动保存带时间戳文件名的 CSV 数据文件 |
| **面形分析** | 基于 Eigen 最小二乘平面拟合，计算 PV、RMS、倾斜角 |
| **PMAC 远程控制** | 内置 Telnet 自动登录与 gpascii 通道，支持手动发送运动指令 |
| **多采样率** | 支持 100/200/400/1000/2000 Hz 采样频率可选 |

## 硬件依赖

### STIL CCS Prima 色散共焦传感器

- 采样率：最高 2000 Hz
- 输出：表面高度（µm）+ 反射光强度（归一化）

- 接口：以太网（TCP/Telnet，默认 IP `192.168.0.200`，端口 `5002`）
- 默认登录：用户名 `root`，密码 `deltatau`
- 使用 Gate3 EQU 硬件比较器输出触发脉冲
- 需配置编码器通道与 EQU 输出通道（具体通道号在代码中硬编码）

## 构建方法

### 前置条件

1. 安装 **Visual Studio 2022**（含 C++ 桌面开发工作负载）
2. 安装 **Qt 6.10.2 MSVC 2022 64-bit**，并安装 **Qt VS Tools** 插件
3. 安装 **STIL CCS Prima 驱动程序**，确保以下文件存在：
   - `D:\STIL\DLL\Headers\Mchr.h`（及同目录其他头文件）
   - `D:\STIL\DLL\Lib_x64\Dll_chr.lib`
4. 确保运行时 `Dll_chr.dll` 在系统 PATH 中或与可执行文件同目录

### 编译步骤

1. 用 Visual Studio 2022 打开 `stil_drive.sln`
2. 选择配置：`Debug|x64` 或 `Release|x64`
3. 生成解决方案（Ctrl+Shift+B）

### 运行时部署（Release）

使用 Qt 提供的 `windeployqt` 工具将所需 Qt DLL 复制到输出目录：
```bat
windeployqt.exe x64\Release\stil_drive.exe
```
`x64\Release\` 目录中已包含一份部署好的 Qt 运行时文件供参考。

1. **连接 PMAC**：在运动控制区填入 IP 和端口，点击"连接 PMAC"，等待 Telnet 自动登录完成（日志中出现 `gpascii` 提示符）

6. **面形分析**：点击"分析"，选择已保存的 CSV 文件，软件输出 PV/RMS/倾斜角

## 扫描模式详解

### 1. X 轴线性扫描

- **运动轴**：X 轴单轴线性扫描
- **触发间隔**：100,000 编码器计数 = 1 µm
- **坐标还原**：`x = pulse_index × 1 µm`，`y = 0`
- **用途**：直线截面轮廓测量

### 2. C+X 轴螺旋扫描

- **运动轴**：C 轴（旋转）+ X 轴（径向进给）联动
- **触发间隔**：640,000 编码器计数 = 0.5°（C 轴）
- **坐标还原**：
  - `angle = pulse_index × 0.5° × π/180`
  - `radius = R₀ - pulse_index × (pitch / 720)`（每转进给螺距）
  - `x = radius × cos(angle)`，`y = radius × sin(angle)`
- **用途**：旋转光学元件全口径面形测量（从边缘向中心螺旋）

### 3. C 轴纬度圆弧扫描

- **运动轴**：C 轴旋转，X 轴固定
- **触发间隔**：40,960 编码器计数 = 0.1°（C 轴）
- **扫描范围**：370°（360° + 10° 重叠）
- **坐标还原**：
  - `angle = pulse_index × 0.1° × π/180`
  - `x = R × cos(angle)`，`y = R × sin(angle)`（R 为固定半径）
- **用途**：球面元件单条纬度圆弧轮廓测量

## 数据格式

CSV 文件以时间戳命名，格式为 `YYYYMMDD_HHMMSS_SyncData.csv`。

### 列格式

| 列 | 内容 | 单位 |
|----|------|------|
| 第 1 列 | X 坐标 | µm |
| 第 2 列 | Y 坐标 | µm |
| 第 3 列 | 表面高度 (altitude) | µm |
| 第 4 列 | 反射光强度 (intensity) | 归一化（0~1） |

### 示例数据

```
0.000,0.000,125.432,0.756
1.000,0.000,125.441,0.761
2.000,0.000,125.438,0.759
...
```

## 面形分析算法

点击"分析"按钮后，软件执行以下步骤：

1. **读取 CSV**：加载所有数据点 `(x, y, z, intensity)`
2. **强度过滤**：仅保留强度在 5%~95% 范围内的点（去除无效测量和饱和点）
3. **最小二乘平面拟合**：
   - 构建设计矩阵 `A`（每行为 `[x, y, 1]`）和目标向量 `b`（高度值）
   - 使用 Eigen `colPivHouseholderQr()` 求解 `Z = ax + by + c`
4. **去除倾斜**：计算每点残差 `residual = z - (ax + by + c)`
5. **输出指标**：
   - **PV (Peak-to-Valley)**：最大残差 - 最小残差（nm）
   - **RMS**：残差均方根（nm）
   - **X 倾斜角**：`arctan(a)` 转换为角秒（arcsec）
   - **Y 倾斜角**：`arctan(b)` 转换为角秒（arcsec）

## PMAC 通信协议

软件通过 `QTcpSocket` 建立 TCP 连接，实现 Telnet 自动登录与命令收发：

### 自动登录流程

1. 连接到 `IP:5002`
2. 自动应答 Telnet IAC 选项协商（WILL/WONT/DO/DONT）
3. 发送用户名 `root\r\n`，等待密码提示
4. 发送密码 `deltatau\r\n`，等待 shell 提示符
5. 发送 `gpascii -2\r\n`，打开 PMAC ASCII 命令通道
6. 登录完成后可发送 PMAC 运动/配置指令

### Gate3 EQU 触发脚本示例（X 轴模式）

```
Gate3[0].Chan[0].CompAdd=100000  // 触发间隔：100000 count = 1µm
Gate3[0].Chan[0].CompA=0
Gate3[0].Chan[0].CompB=99999
Gate3[0].Chan[0].Equ1Ena=1      // 使能 EQU 输出
```

## 关键参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| PMAC IP | `192.168.0.200` | PMAC 控制器以太网地址（可在 UI 中修改） |

| 起始半径 | `10000 µm` | 螺旋/纬度扫描的起始径向位置 |

| C 轴螺旋触发间距 | `640,000 count` = 0.5° | 螺旋扫描角度分辨率 |
| C 轴纬度触发间距 | `40,960 count` = 0.1° | 纬度扫描角度分辨率 |
| 缓冲区大小 | 2 × 1 点双缓冲 | STIL DLL TRE 模式每脉冲 1 点 |

## 已知限制

1. **STIL DLL 路径硬编码**：STIL SDK 路径固定为 `D:\STIL\DLL\`，迁移到其他机器时需修改 `.vcxproj` 中的 include/lib 路径
2. **PMAC 通道硬编码**：Gate3 通道号（`Gate3[0].Chan[0]`）等硬编码在代码中，不同机器配置需修改源码
3. **仅支持 x64 Windows**：依赖 Win32 API 和 STIL x64 DLL，不支持其他平台
4. **单传感器**：每次只能连接并使用一个 STIL 传感器（取 USB 设备列表第一项）
5. **软件停止无位置反馈**：自动停止条件（如到达目标点数）依赖软件计数，不做硬件位置核验
